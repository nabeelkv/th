Welcome to the Front End Techdegree

We're excited that you've decided to join Treehouse on this new journey toward becoming a Front End Web developer. This Techdegree will train you in the languages you'll use — HTML, CSS and JavaScript — as well as the skills you'll need to build web sites.

Along the way, you'll build 9 real-world projects, including complex page designs that morph to fit different size devices like a mobile phone, a tablet, or a large monitor. You'll learn how to turn drab text into colorful, and complex layouts that are fun to read and look at.

You'll also build interactive web pages using JavaScript, the most popular programming language in the world. You'll build an interactive photo gallery, a video player complete with speed control, closed captioning and communicate with other websites using JavaScript.

Finally, you'll build a portfolio to showcase your skills, as well as the projects you've created as part of the Techdegree.

Welcome, and have fun with Front End Web Development!
